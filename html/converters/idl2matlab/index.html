idl2matlab: IDL to Matlab/Scilab translator

version 1.6.1 - Sept 2015 (c) ILL
IDL and Matlab are two major numerical languages for scientists. Additionally, Scilab is a free Matlab-like software, which integrates a high compatibility (syntax, functions, graphics, user interfaces).
Many IDL programs have been developed in large facilities studying astrophysics, geology,medicine, particle and nuclear physics.
On the other hand, Matlab is widely used in smaller laboratories (universities, national scientific agencies, ...).
In order to extend the portability of such IDL programs, we have developed an automatic translator from IDL to Matlab and Scilab, idl2matlab.
INPUT: idl2matlab
 OUTPUT
An IDL code (file *.pro)
 The corresponding Matlab/Scilab code.

The result will be proposed as a ZIP archive holding the translated code. To run it you will need to install our idl2matlab library and initialise it with a call to 'i2m_init' (for Matlab), or 'i2s_init' (for Scilab). Get idl2matlab-lib.

Limitations

The translator is not perfect, and may fail for unsupported IDL grammar. Some functions are not translated. The translation does NOT support objects and complex plotting routines.

How it works

The IDL code is sent to the idl2matlab executable which is built using a LeX/Yacc grammar analyser. The code is then written into a target language (here Matlab or Scilab). No information is kept on the server, files are deleted after the conversion.
