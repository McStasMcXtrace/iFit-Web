cif2hkl: CIF to F2(HKL) calculator

version 1.2 - March 2016 (c) ILL

The CIF2HKL tool allows to compute the structure factors of powders and single crystal materials. It requires as input a crystallographic information, and provides a list of F²(HKL) to be used for e.g. McStas and McXtrace components such as PowderN, Single_crystal and Isotropic_Sqw.

INPUT: cif2hkl
 OUTPUT
A CIF/ShelX/FullProf file
 A file with the F²(HKL) reflection list for use with e.g.  McStas and McXtrace (Laz/Lau)

Limitations

It is a good idea to check the Physical parameters pre-computed in the header of the HKL file, especially the cross sections, the density and molar weight.

How it works

The cif2hkl program calls subsequently the ReadN_set_Xtal_Structure, Get_MaxNumRef,  Hkl_Uni or Hkl_Gen_SXtal, and Structure_Factors from the CrysFML [repository]. No information is kept on the server, files are deleted after the conversion.
