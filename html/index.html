e-Computing services

E. Farhi, Aug 2019

We provide a set of web infrastructure tools to:
  • convert file within formats of use for neutron scattering
  • compute phonon dispersions in 4D
  • use cloud services (virtual machines, Jupyter notebooks)
  • estimate the measurement from a neutron scattering experiment

All this software can also be installed on your own computer or on a cluster, to be used as web services, or as commands. Refer to the Installation page for more.

Limitations

These tools are provided "as is", and should be considered experimental. If this works for you, that's good. If it fails, please report to the email address indicated in the footer below.

How it works

All these services are simple scripts that get parameters from web pages, execute dedicated commands, and return information upon completion. Most of this work is derived from the iFit infrastructure.
