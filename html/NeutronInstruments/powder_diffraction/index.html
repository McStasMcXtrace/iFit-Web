powder diffraction (neutron scattering)

version 1.9 - E. Farhi, March 2017

The powder diffraction tool is a tool to compute the powder neutron diffractogram from a material specification, and some limited instrument configuration parameters. It is based on the iFit/McStas interface. The model is a constant wavelength source, suited for e.g. continuous sources.

The model is built by assembling:
  1. a neutron source (continous)
  2. a 1st collimator
  3. a monochromator
  4. a 2nd collimator
  5. a powder sample (cylindrical)
  6. a 3rd radial collimator
  7. a radial detector ("banana" shaped)

You may also try the e-neutrons powder diffractometer simulator at https://www.e-neutrons.org/mcstas-distrib (requires a login at e-neutrons.org).

1- Material structure file to Upload (max 5Mb):

Supported formats:

  • CIF (.CIF) from e.g. the Inorganic Crystal Structure Database (ICSD) and The Crystallography data base
  • FullProf (.CFL) for instance the Na2Ca3Al2F14.
  • FullProf (.PCR)
  • ShelX (.SHX)
  • ShelX (.INS)
  • ShelX (.RES)

2- Indicate the incident neutron wavelength to use (lambda Î» in [Angs]):
The 'hot' neutrons are basically below lambda=1 Angs, thermal neutrons are from 1 to 3. If unsure, use 1.2 or 2. If the desired wavelength is not in the list, use the closest available choice.


3- Indicate the monochromator d-spacing to use (DM, in [Angs]):
Select one of the above lattice spacing to use. For unlisted materials, use the closest d-spacing. If unsure, use PG(002) or Cu(220)

4- Indicate if the monochromator is flat or curved (RM):
A flat monochromotor provides the best resolution, but lowest flux at the sample. Most neutron diffractometers use a vertical focusing geometry, which does not affect the horizontal resolution in the scattering plane. Double focusing monochromators bring the maximum flux, but a degraded resolution. If unsure, use vertical focusing only.

5- Indicate the monochromator mosaic (ETA, in [arc-min]):
Good monochromator crystals have a small mosaicity (e.g. less than 5 arc-minutes). Highly oriented pyrolythic graphite (PG) have usually a much higher mosaicity, e.g. 30 arc-minutes. [1 degree = 60 arc-minutes]. If unsure, use 15-20.

6- Indicate the source/guide - monochromator distance (L1, in [m]):
This is the distance between the neutron source, and the monochromator. If at the end of a guide or unsure, use e.g. 10 meters.

7- Indicate the monochromator - sample distance (L2, in [m]):
This is the distance between the monochromator and the sample. If unsure, use e.g. 2 meters.

8- Indicate the sample - detector distance (L3, in [m]):
This is the distance between the sample and the detector. If unsure, use e.g. 1.5 meters.

9- Indicate the source/guide - monochromator collimation (ALPHA1):
The 1st neutron collimator is located before the monochromator. If unsure use 5 arc-minutes divergence. A value of 60 does not affect the neutron beam. Use 0 to remove the collimator.

10- Indicate the monochromator - sample collimation (ALPHA2):
The 2nd neutron collimator is located between the monochromator and the sample. If unsure use 5 arc-minutes divergence. A value of 60 does not affect the neutron beam. Use 0 to remove the collimator.

11- Indicate the sample - detector collimation (ALPHA3):
The 3rd radial neutron collimator is located between the sample and the detector. If unsure use 5 arc-minutes divergence. A value of 60 does not affect the neutron beam. Use 0 to remove the collimator.

12- Indicate the sample diameter (WS, in [cm]):
The sample diameter plays a significant role on the resolution. Large samples give more scattering intensity, but degrade the resolution. Small samples scatter less, and may be harder to obtain.

What will happen

The input material description will be processed and sent to cif2hkl (a conversion tool based upon CrysFML). The resulting powder file (F2(HKL) reflection list) is sent to the templateDIFF McStas instrument. Results are then imported by iFit, and displayed.

Limitations

This simulation should be considered as a fast powder diffractogram evaluation. For security reasons, only discrete settings are available in this service.
