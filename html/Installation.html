Installation


This service can be installed on any computer.

Requirements

We recommend to install this service on a computer with e.g. 16 Gb memory, 8-32 cores, and a 1 Tb disk. A smaller disk is possible, but some of the services store the results locally (e.g. sqw_phonons, ...). The installation of the base infrastructure requires about 2 Gb on disk (mostly DFT codes and pseudo-potentials).

Installation

Install a Debian/Ubuntu/Mint system on a computer. You can for instance get the Ubuntu 14.04 ISO. The installation of this system can equally be done on a virtual machine, for instance using VirtualBox.

Once set-up and running, issue the following commands in a terminal. For this you need the super-user (administrator) rights.
sudo apt-add-repository 'deb http://packages.mccode.org/debian stable main'
sudo apt-get update
sudo apt-get install ifit-web-services

Then, it is advisable that you configure the services by editing the files (search for 'configuration'):

  • /usr/lib/cgi-bin/computing_sqw_phonons.pl

Then you should be able to launch the services at e.g. http://localhost/ifit-web_services or the [machine IP or URL])/ifit-web_services address.
