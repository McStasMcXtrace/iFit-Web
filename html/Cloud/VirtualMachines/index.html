Freshly Cooked Virtual Machines

version 1.0 - Aug 2019 (c) SOLEIL

This page allows to select and launch a Virtual machine in a Browser Window, with distant execution. This way, you can access pre-installed systems with specific applications matching specific scientific domains.
INPUT: VirtualBox machine
 OUTPUT
Select the machine you need
 A browser showing the distant machine through VNC

1- Indicate the machine you need:

  • Lubuntu: a minimal Lubuntu 18.04 system (used for testing).

  A virtual machine will be created, and a browser window will be opened.

Availability and Credits

If you use this software, please acknowledge

Limitations

The virtual machines are made secure by inactivating some of their functionalities.

How it works

A one-shot virtual machine is cloned from a template. It is then started with QEMU/KVM. A VNC client (noVNC) is connected to it, and shown in your browser. All the available virtual machines are stored on our Apache server.
